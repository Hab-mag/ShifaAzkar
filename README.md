# Шифа — Аяты, Азкары и Дуа

Мобильное приложение с аятами лечения (рукъя), азкарами и дуа по разным случаям жизни.

## Содержание

- **Аяты лечения (Рукья)** — Аль-Фатиха, Аят аль-Курси, конец Аль-Бакара; суры от сглаза и колдовства (Аль-Ихляс, Аль-Фаляк, Ан-Нас); аяты при болезни.
- **Азкары** — утренние и вечерние поминания, после намаза, перед сном.
- **Дуа** — при печали и тревоге, об исцелении, защите, повседневные (дом, еда и т.д.).

Приложение **полностью офлайн**: нет запросов в интернет, все тексты и стили загружаются с устройства — открывается быстро.

## Как открыть на телефоне

1. **Через браузер**  
   Залейте папку `ShifaAzkar` на любой хостинг или откройте файл `index.html` локально (например, через приложение «Файлы» и браузер).

2. **Добавить на главный экран (PWA)**  
   - Откройте сайт в Chrome/Safari на телефоне.  
   - В меню выберите «Добавить на главный экран» / «Add to Home Screen».  
   - Иконка «Шифа» появится на рабочем столе как приложение.

Для установки как PWA нужны иконки `icon-192.png` и `icon-512.png` в папке приложения (при их отсутствии «Добавить на главный экран» всё равно может работать, но без своей иконки).

## Запуск на компьютере

Дважды откройте `index.html` в браузере или запустите локальный сервер в папке проекта, например:

```bash
cd ShifaAzkar
npx serve .
```

После этого откройте в браузере указанный адрес (например, http://localhost:3000).

## Структура

- `index.html` — разметка и экраны.
- `styles.css` — стили под мобильные устройства.
- `data.js` — данные: аяты, азкары, дуа по категориям.
- `app.js` — навигация и просмотр текстов.
- `manifest.json` — настройки PWA для установки на телефон.

## Сборка APK для Android (быстрая загрузка офлайн)

APK ставится на телефон как обычное приложение: открывается мгновенно, работает без интернета.

**Что нужно на компьютере:**
- [Node.js](https://nodejs.org/) (LTS)
- [Android Studio](https://developer.android.com/studio) (для сборки APK)

**Шаги:**

1. Откройте папку `ShifaAzkar` в терминале (или командной строке).

2. Установите зависимости и добавьте Android-проект:
   ```bash
   npm install
   npx cap add android
   ```

3. Синхронизируйте файлы приложения в Android-проект:
   ```bash
   npx cap sync android
   ```
   (Файлы приложения для APK лежат в папке `www`. Если правили файлы в корне — скопируйте `index.html`, `styles.css`, `app.js`, `data.js`, `manifest.json` в `www` перед этой командой.)

4. Откройте проект в Android Studio и соберите APK:
   ```bash
   npx cap open android
   ```
   В Android Studio: **Build → Build Bundle(s) / APK(s) → Build APK(s)**.  
   Готовый APK будет в `android/app/build/outputs/apk/debug/app-debug.apk`.

5. Перенесите `app-debug.apk` на телефон (USB, мессенджер, облако) и установите. При первом запуске может понадобиться разрешить установку из «Неизвестных источников».

## APK без Android Studio (ноутбук слабый)

**Вариант 1 — PWA (без APK):**  
Откройте приложение на телефоне в Chrome (с сайта или скопируйте папку на телефон и откройте `www/index.html` в браузере). Меню Chrome → «Добавить на главный экран». Иконка «Шифа» появится на рабочем столе, работает как приложение и офлайн.

**Вариант 2 — APK через GitHub (сборка в облаке):**
1. Зарегистрируйтесь на [GitHub](https://github.com), установите [Git](https://git-scm.com).
2. Создайте новый репозиторий на GitHub, загрузите туда папку `ShifaAzkar` (все файлы, включая папки `.github` и `www`).
3. В репозитории откройте вкладку **Actions** → запустите workflow **Build APK** (или он запустится сам при пуше в `main`/`master`).
4. Когда сборка закончится (зелёная галочка), откройте этот запуск → внизу блок **Artifacts** → скачайте **Shifa-APK** (внутри архив с `app-debug.apk`).
5. Перенесите APK на телефон и установите.

Сборка идёт на серверах GitHub, Android Studio на ноутбуке не нужна.
